# TalentMatch AI: Intelligent Resume Screening Platform

TalentMatch AI is a hackathon-ready platform that uses NLP and Machine Learning to revolutionize the recruitment process. It doesn't just match keywords; it understands skills, identifies gaps, and provides an automated learning roadmap for candidates.

## 🚀 Features
- **Semantic Matching**: Uses TF-IDF and Cosine Similarity to score resumes against job descriptions.
- **Skill Extraction**: Rule-based NLP using spaCy to identify technical proficiency.
- **Gap Analysis**: Instantly highlights missing skills required for the role.
- **AI Roadmap**: Generates a step-by-step guided path for candidates to bridge their skill gaps.
- **Premium UI**: Glassmorphism design with interactive charts and fluid animations.

## 🛠 Tech Stack
- **Backend**: FastAPI, spaCy, Scikit-learn, PyPDF2
- **Frontend**: Vanilla HTML5, Vanilla CSS3, Vanilla JavaScript, Chart.js, Lucide Icons
- **Language**: Python 3.8+, Node.js 16+

## ⚡ Quick Start

### 1. Backend Setup
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### 2. Frontend Setup
Simply open `frontend/index.html` in your browser.
```bash
cd frontend
npx serve .
```

### 3. Usage
1. Open the frontend URL (usually `http://localhost:5173`).
2. Upload a resume (PDF/DOCX).
3. Paste a Job Description.
4. Click **Analyze Matching**.
5. Explore the match score, skill gaps, and the personalized learning roadmap.
